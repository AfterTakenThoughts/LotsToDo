# LotsToDo

LotsToDo is a (as of now) extremely simple To Do list app to track your tasks. For now, you can add, remove, and save the tasks for later use.

# Note

This project is primarily made for myself, so much of the (planned) features are made for my purposes. That being said, if you find some To Do apps cumbersome to use, it might be worth using when LotsToDo matures enough!

Some planned features include tags, To Do file parsers (Todo.txt), folder parser, and many more.

## Installation

Binary releases will come later, when more features on the app are implemented. For now, you can use the source code to run the app.

## Contributing

The app is very WIP, and there are a lot of planned features that are not implemented. So as of now LotsToDo can't accept pull requests or any contributions from anyone.
